# Zoheb Khan / Centauri Operating Repository

This repository is a public-safe operating workspace for Centauri-related documentation systems, AI-agent workflows, Codex tasks, product/profile scaffolds, and execution checklists.

> Public repository notice: do **not** commit confidential investor materials, legal documents, private financial models, unreleased technical data, proprietary partner information, personal data, credentials, API keys, contracts, or restricted documents. Keep sensitive work in private systems only.

## Current Purpose

- Establish a GitHub workspace that ChatGPT, Codex, and human reviewers can reference.
- Track public-safe project structure, tasks, and execution frameworks.
- Maintain agent instructions for consistent reviews, documentation, and future code work.
- Create a clean base for product, solar, AI operations, and corporate documentation tooling.

## Operating Principles

1. Keep public materials factual, neutral, and verifiable.
2. Separate public-safe documentation from confidential business records.
3. Use issues for work intake and pull requests for reviewable changes.
4. Never add credentials, personal data, private legal content, or proprietary material.
5. When a task requires confidential inputs, reference the need for private review instead of storing the data here.
