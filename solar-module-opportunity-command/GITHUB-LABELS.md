# Recommended GitHub Labels — Solar Module Opportunity Pipeline

Create or maintain these labels in the repository for opportunity workflow control.

## Source Labels

| Label | Purpose |
|---|---|
| source:samgov | Federal SAM.gov opportunity |
| source:doe | DOE / federal energy funding opportunity |
| source:utility | Utility or IPP procurement opportunity |
| source:epc | EPC or developer opportunity |
| source:distributor | Distributor or wholesale channel opportunity |
| source:direct-buyer | Private direct buyer opportunity |

## Type Labels

| Label | Purpose |
|---|---|
| type:module-supply | Solar module supply opportunity |
| type:solar-storage | Solar plus BESS or microgrid opportunity |
| type:funding | Grant, FOA, loan, or manufacturing funding opportunity |
| type:framework | Long-term framework supply agreement |

## Status Labels

| Label | Purpose |
|---|---|
| status:intake | New opportunity intake |
| status:qualified | Qualified for bid/no-bid review |
| status:pricing | Pricing required |
| status:proposal | Proposal package in development |
| status:risk-review | Compliance/legal/risk review active |
| status:ready-approval | Ready for executive approval |
| status:submitted | Submitted to buyer |
| status:award-pending | Awaiting award decision |
| status:awarded | Awarded or PO received |
| status:no-bid | Not pursued |
| status:lost | Lost opportunity |

## Priority Labels

| Label | Purpose |
|---|---|
| priority:p1 | Urgent / high-value / deadline-sensitive |
| priority:p2 | Important but manageable |
| priority:p3 | Monitor or low urgency |

## Risk Labels

| Label | Purpose |
|---|---|
| risk:compliance | Compliance risk |
| risk:delivery | Delivery or lead-time risk |
| risk:legal | Legal or contract risk |
| risk:pricing | Price, margin, or payment-term risk |
| risk:technical | Product, certification, datasheet, or claim risk |

## Special Control Labels

| Label | Purpose |
|---|---|
| dc-required | Domestic content required |
| exec-review | Executive review required |
| bafo | Best and final offer active |
| q-and-a | Buyer Q&A / clarification active |
| blocker | Blocking issue |
