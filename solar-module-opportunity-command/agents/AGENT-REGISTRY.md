# Agent Registry — Solar Module Opportunity Command

## Command Layer

| Agent | Mission | Key Decisions |
|---|---|---|
| Solar Supply Opportunity Command Agent | Own the full pipeline from detection to award handoff | Status movement, bid/no-bid, escalation, executive approval |
| Capture Strategy Agent | Define sector strategy, win themes, buyer priority, and strategic value | Pursuit priority and value narrative |
| GitHub Operations Agent | Maintain issues, templates, labels, checklists, and project board discipline | Workflow hygiene and repository controls |

## Monitoring Layer

| Agent | Sources | Output |
|---|---|---|
| SAM.gov Monitoring Agent | Federal contract opportunities, RFIs, RFQs, RFPs, IFBs | Federal intake issue |
| DOE Funding Agent | DOE, MESC, EERE, manufacturing and supply-chain funding | Funding opportunity brief |
| Utility / IPP Procurement Agent | Utility and IPP procurement portals | Utility pursuit card |
| EPC / Developer Demand Agent | EPCs, developers, contractors, solar procurement teams | Direct sales target brief |
| Distributor Channel Agent | Distributors, wholesalers, installer networks | Channel quote package |
| Strategic Buyer Agent | Data centers, defense, microgrids, federal facilities | Strategic pursuit brief |

## Qualification Layer

| Agent | Mission | Output |
|---|---|---|
| Qualification Agent | Score opportunity fit, urgency, margin, risk, and strategic value | Bid/no-bid scorecard |
| Product Fit Agent | Match buyer requirement to module family and datasheet | Product fit matrix |
| Supply Chain Agent | Confirm inventory, production slot, lead time, shipping, and delivery path | Supply confirmation memo |

## Execution Layer

| Agent | Mission | Output |
|---|---|---|
| Pricing Agent | Build price per watt, freight, terms, discount, and margin | Pricing approval sheet |
| Compliance / Risk Agent | Review technical, domestic-content, legal, contract, and buyer requirements | Compliance and risk checklist |
| Proposal Agent | Build response package and supporting attachments | Proposal package |
| Graphics / Datasheet Agent | Prepare product visuals, datasheets, supply maps, and branded proposal graphics | Visual package |
| Submission Agent | Handle portal upload, email submission, confirmation, and deadline discipline | Submission receipt record |

## Post-Submission Layer

| Agent | Mission | Output |
|---|---|---|
| Q&A / Clarification Agent | Manage buyer questions and technical clarifications | Q&A response log |
| BAFO Agent | Manage best-and-final pricing and commercial updates | BAFO package |
| Award Handoff Agent | Transfer won deals to operations, supply chain, legal, and finance | Award handoff memo |
| Lessons Learned Agent | Capture why opportunity was won, lost, or no-bid | Lessons learned note |

## Agent Handoff Rules

| From | To | Required Handoff Material |
|---|---|---|
| Monitoring Agent | Command Agent | Source link, buyer, deadline, notice type, summary |
| Command Agent | Qualification Agent | Intake issue and initial triage notes |
| Qualification Agent | Product Fit Agent | Scorecard and buyer technical requirements |
| Product Fit Agent | Supply Chain Agent | Matched module family, wattage, required volume |
| Supply Chain Agent | Pricing Agent | Supply availability, freight assumptions, lead time |
| Pricing Agent | Compliance / Risk Agent | Price sheet, assumptions, terms, exceptions |
| Compliance / Risk Agent | Proposal Agent | Compliance checklist and redlines/exceptions |
| Proposal Agent | Submission Agent | Final package draft and required buyer forms |
| Submission Agent | Post-Submission Agent | Confirmation receipt, Q&A deadlines, buyer contact |
