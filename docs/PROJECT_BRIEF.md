# Project Brief (Public-Safe)

## Overview

This repository serves as a structured, public-safe workspace for organizing system-level documentation, execution frameworks, and AI-assisted workflows related to Centauri-style operations, product development planning, and engineering documentation.

## Goals

- Provide a structured GitHub-based execution system for documentation and planning.
- Enable Codex / AI-assisted agents to operate in a controlled, auditable workflow.
- Maintain separation between public documentation and private business logic.
- Support scalable project decomposition into issues and tasks.

## Scope

### Included

- Documentation frameworks
- Task tracking via GitHub Issues
- AI agent operating instructions
- Public-safe system design
- High-level architecture planning

### Excluded (Strict)

- Investor materials
- Financial models
- Legal agreements or contracts
- Private technical implementations
- Credentials, keys, tokens
- Any proprietary or confidential data

## System Behavior

The repository operates as a:

- Documentation hub
- Task orchestration layer
- AI agent execution scaffold
- Reviewable workflow system

## Key Principles

1. Transparency over assumptions
2. Modularity over monoliths
3. Public safety over completeness
4. Reviewability over speed
5. Separation of concerns (public vs private systems)

## Outputs

- Markdown documentation
- GitHub issues for execution tracking
- Pull requests for changes
- Structured backlog for planning
